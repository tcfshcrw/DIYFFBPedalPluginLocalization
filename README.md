### How to Use Localization

1. **Verify SimHub Version**: Ensure that your SimHub version is **9.11.0 or later**.
2. **Download Language Files**: Download `DiyActivePedal.resx` and the `.resx` file for your specific language.
3. **Install Files**: Place both files into the `SimHub\Languages` directory.
4. **Enable Localization**: Open SimHub, navigate to **Settings**, and select your target language.

### How to Contribute Translations

1. **Open SimHub** and navigate to **Settings**.
2. Select your desired language from the list.
3. Click the **Edit translation** button to enter the translation mode.
4. Translate the relevant fields for this plugin and save your changes.
5. Navigate to your SimHub installation folder and open the `languages` subdirectory.
6. Locate the `.resx` file corresponding to the language you just edited.
7. **Fork** this repository on GitHub.
8. Copy the updated `.resx` file into the corresponding folder in your forked repository.
9. Submit a **Pull Request (PR)** to merge your translation.
